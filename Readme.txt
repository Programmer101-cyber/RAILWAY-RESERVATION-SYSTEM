Thank you for using the Railway Reservation Version 1.0.1.
-----------------------------------------------------------
The Program contains two modes :

1)User Mode 
2)Admin Mode 

User mode lets the user book and cancel ticket.

The default password of Admin Mode is "pass".
In the admin mode,the use can add,delete train details from the database.
In addition to that there is an option to change the password of the admin mode.
